# 🚀 Udyog Saarthi

**Udyog Saarthi** is an inclusive job coaching and skill development platform designed to empower adults with disabilities and individuals eligible under the 4% reservation scheme. The platform provides structured learning resources, career guidance, and employment support to help users develop workplace-ready skills and achieve greater independence.

## 🌟 Features

* 📚 **Job Coaching Modules**

  * Soft Skills Training
  * Resume Writing
  * Interview Preparation
  * Digital Literacy

* 🎯 **Career Development**

  * Step-by-step guidance for employability
  * Practical workplace readiness resources
  * Accessible learning content

* 🤝 **Inclusive Design**

  * Developed with accessibility and usability in mind
  * Supports diverse learning needs

* 🌐 **Responsive Web Application**

  * Works across desktop and mobile devices

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
* **Backend:** Flask
* **Database:** SQLite / MySQL
* **Version Control:** Git & GitHub

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/Udyog-Saarthi.git
cd Udyog-Saarthi
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://localhost:5000
```

## 🎯 Vision

Udyog Saarthi aims to bridge the gap between skill development and employment opportunities by providing accessible and inclusive career support for persons with disabilities. The project seeks to promote independence, confidence, and equal opportunities in the workforce.
